# brott-studio.github.io

Public website for **Brott Studio**, a 100100 Productions venture.

Live: https://brott-studio.github.io

Built with [Astro](https://astro.build/), deployed via GitHub Pages from `main`.

## Dev

```bash
npm install
npm run dev       # http://localhost:4321
npm run build     # dist/
```

## Deploy

GitHub Actions builds on push to `main` and deploys to Pages automatically.
